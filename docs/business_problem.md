# Business Problem

Many EdTech users install or sign up but do not complete their first successful learning session. In a doubt-solving app, the first completed session is a critical moment because it proves that the product can connect a student with useful academic help quickly.

The project analyzes where users drop off across the journey:

1. Install or signup
2. Onboarding
3. First question submission
4. Tutor search and connection
5. Session completion
6. Repeat usage
7. Payment conversion

The goal is to identify the most important product bottlenecks and recommend actions that improve activation, retention, and monetization.

All data is synthetic and generated for portfolio use only.
